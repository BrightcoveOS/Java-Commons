About

This goal of this project is to provide common libraries needed for other Open Source @ Brightcove Java projects.

Currently the vast majority of these libraries are present simply to support the Brightcove Java Media API Wrapper, but other libraries will be added for other projects (e.g. to assist in creating Brightcove Batch Manifest files).

Requirements

Whenever possible, the libraries will be compatible with Java 1.5, but in some cases Java 1.6 may be required. Each commons library should make note of this.
